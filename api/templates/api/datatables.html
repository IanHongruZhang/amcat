Limit to results.

Exporting to SPSS has certain limitations and will therefore not transfer information perfectly.


  • 1. Each cell cannot contain more than 32767 characters;
  • 2. Tabs are removed and replaced by spaces;
  • 3. Newlines are removed and replaced by dots.

These limitations are inherent to SPSS itself, and cannot be solved in AmCAT. If you see behaviour which cannot be explained by the limitations listed above, feel free to file a bug report at github.com/amcat/amcat.
